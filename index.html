Cómo añadir música

Guía para GitHub Pages

Como esta es una página estática en GitHub Pages, sigue estos pasos para añadir música:

  1. Sube tu archivo .mp3 y la imagen a la carpeta music/ en GitHub.
  2. Abre el archivo songs.js en GitHub.
  3. Añade un nuevo objeto a la lista STATIC_SONGS.
  4. Guarda los cambios y la página se actualizará sola.
